Developed CosmicChat – an AI-powered chat application using Next.js, integrating Google's Generative AI for intelligent, context-aware conversations and secure content filtering for safe user interactions.

Engineered a dynamic, responsive UI – built with Radix UI and styled with Tailwind CSS, offering seamless functionality across devices with customizable light/dark themes for enhanced user personalization.

Built a scalable architecture – utilizing modular components and Context API for efficient state management, ensuring maintainability and performance optimization across the application.
